# Livraria Casa dos Contos

## *Website*

## Para executar o website

1. Instalar modules NPM

      ```
      $ npm install
      ```

2. Iniciar servidor de desenvolvimento

      ```
      $ gulp dev
      ```

3. Criar distribuição do website

      ```
      $ gulp dist
      ```

4. Rodar servidor backend

      ```
      $ npm start
      ```
